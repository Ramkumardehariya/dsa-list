## Enable GitHub Pages

To use the interactive HTML version with links that open in new tabs:

1. Go to your repository on GitHub
2. Click **Settings** tab
3. Scroll down to **Pages** section
4. Under **Source**, select **Deploy from a branch**
5. Select **main** branch and **/(root)** folder
6. Click **Save**
7. Wait a few minutes, then visit: `https://ramkumardehariya.github.io/dsa-list/`
